下面就介绍下Markdown的一些基本语法：
一、标题：
标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 #号即可。
# 一级标题
## 二级标题
### 三级标题
###### 六级标题
#号后需要加一个空格，一共有6级，一个#对应一级，下面是效果：
一级标题
二级标题
三级标题
六级标题
二、引用：
当你想引用别处的句子时，一般会使用引用格式，Markdown里的引用格式就是在句子前加个>大于号。
> 冯潇霆: 路还长，我们一起走。
>后也是有空格，效果如下:
冯潇霆: 路还长，我们一起走。
注：
1.如果想嵌套引用，则在引用体中用不同数量的>来实现嵌套引用；
2.如果你在引用体后面直接换行输入，那么该行还会被纳入引用体，因此你需要输入两次换行达到脱离引用；
三、列表：
列表有三种情况，有序列表、无序列表和层级列表，有序列表只需前面加上- 或者*后面接个空格：
1、无序列表：
- 首页 或 * 首页
- 文件 或 * 文件
- 编辑 或 * 编辑
- 帮助 或 * 帮助
效果如下：
?	首页
?	文件
?	编辑
?	帮助
2、有序列表：
有序列表则直接在文字前加一、 二、 三、 或者其他的序列格式1. 2. 3.：
1. 首页
2. 文件
3. 编辑
4. 帮助
显示如下:
1.	首页
2.	文件
3.	编辑
4.	帮助
3、层级列表：
层级列表一般只支持三级，第一级实心圆点，第二级空心圆点，第三级以后都是实心方点。要实现层级列表，每下一级在开头都要比上一级多输入2个Space或者1个Tab，而且第一级前面不要超过3个Space:

?	一级列表 
o	二级列表 
?	三级列表
四、斜体、粗体：
当我们需要特别强调的时候，往往会使用粗体或者斜体显的醒目一点，在Markdown中，使用一个*包裹就是斜体，用两个*就是粗体。
*我是斜体*
**我是粗体**
效果如下:
我是斜体
我是粗体

[]()

七、换行、分割线：
Markdown换行符有两种形式：
1. </br>
2. <br />
“/br”是连写的，而“br /”之间有一个空格
比如：
如今，邮政的寄件效率已经大大提升，EMS:3天，普快:一周，平邮:半月。</br>当然不过也有例外，有网友在Twitter晒图声称，最近自己收到了3年前的邮政EMS包裹，两部iPhone 5。
当没空格符号时两句话就是连着，但插入换行符时，就会换行，如下：

八、代码块：
如果你是开发人员，代码块肯定不会少，若是我们需要引用代码，我们使用代码框把代码给框起来，使读者更清楚代码的书写，代码框的语法是用两个`包裹就行，比如：
if (textField.text.length >= 6 && textField.text.leng <= 12);
上面代码就是用两个`包裹“if (textField.text.length >= 6 && textField.text.leng <= 12);”的效果；但有些时候我们不止圈住这一行代码，需要圈住更多行，整体一个框的感觉，这时我们需要用到两个```包裹，就是前后连着各写三个`符号,如下：
+(long long)fileSizeWithPath:(NSString *)filePath{
    
    NSFileManager *manager = [NSFileManager defaultManager];
    
    if ([manager fileExistsAtPath :filePath]){
        
        return [[manager attributesOfItemAtPath :filePath error : nil ] fileSize];
    }
    return 0 ;
    
}
